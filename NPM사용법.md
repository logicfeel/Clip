# NPM 사용법 

> NPM  3.10.10 기준
> 문법 : npm <cmd> [args]


NPM 은 해당 경로에 package.json 여부가 중요함


## install : 설치

> npm install 

+ npm install <모듈명> : NPM 배포된 모듈, 현재위치

+ npm install -g <모듈명> : NPM 배포된 모듈, 전역
    * 전역위치 : C:\Users\Young\AppData\Roaming\npm\node_modules

+ npm install https://github.com/logicfeel/Clip.git@<태그 | 버전>
